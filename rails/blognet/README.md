# Blognet - Multi-Blog Platform
**Version:** 8.0.0
**Stack:** Rails 8 + Solid Stack + Falcon

**Port:** 51673

**Domains:** 6 independent blogs

## Hosted Blogs
1. **foodielicio.us** - Food & recipes
2. **stacyspassion.com** - Lifestyle

3. **antibettingblog.com** - Gambling awareness

4. **anticasinoblog.com** - Casino harm reduction

5. **antigamblingblog.com** - Addiction resources

6. **foball.no** - Norwegian football

## Features
- Multi-tenant architecture
- SEO optimized (meta, sitemaps, structured data)

- Threaded comments with moderation

- Media library with CDN

- RSS/Atom feeds per blog

- Custom themes per blog

- Analytics dashboard

# Marketplace - Local Trading Platform

**Version:** 8.0.0  
**Stack:** Rails 8 + Solid Stack + Falcon  
**Port:** Shared with brgen (subdomain)  
**Domain:** markedsplass.brgen.no (+ all city domains)

## Overview

City-based marketplace for buying, selling, and trading goods locally. Part of the Brgen ecosystem.

## Features

- **Local listings** - Buy/sell in your city
- **Categories** - Electronics, furniture, vehicles, etc.
- **Search & filters** - Find exactly what you need
- **Messaging** - Negotiate with sellers
- **Reviews & ratings** - Build trust
- **Photo uploads** - Multiple images per listing
- **Price tracking** - See market trends
- **Saved searches** - Get notified of new listings
- **Safe meet-ups** - Public location suggestions

# Dating - City-Based Dating Platform

**Version:** 8.0.0  
**Stack:** Rails 8 + Solid Stack + Falcon  
**Port:** Shared with brgen (subdomain)  
**Domain:** dating.brgen.no (+ all city domains)

## Overview

Location-based dating platform integrated with Brgen's city network. Match with people in your city.

## Features

- **Smart matching** - ML-powered compatibility
- **Radius filtering** - Find matches nearby
- **Swipe interface** - Quick decisions
- **Real-time chat** - Message your matches
- **Interest-based** - Connect over shared hobbies
- **Safety features** - Verification and reporting
- **Event integration** - Meet at local events
- **Video dates** - Built-in video chat

cd /home/dev/rails

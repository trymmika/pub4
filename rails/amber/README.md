# Amber - AI Fashion Assistant

**Version:** 8.0.0  
**Stack:** Rails 8 + Solid Stack + Falcon  
**Port:** 42189  
**Domain:** amberapp.com

## Overview

Amber is an AI-enhanced social network for fashion. Organize your wardrobe, get style suggestions, and connect with fashion enthusiasts.

## Features

### Wardrobe Management
- Photo-based clothing catalog
- Smart organization and tagging
- Cost-per-wear analytics
- Usage tracking

### Style Assistant
- Daily outfit suggestions
- Mix and match combinations
- Weather-appropriate recommendations
- Event-specific styling

### Social Features
- Fashion feed with trends
- Live chat and webcam streaming
- Public chatrooms
- Outfit feedback from community

### Shopping
- Find pieces that match your wardrobe
- Shop smarter with AI recommendations
- Track purchases and budget

## Quick Start

```zsh
cd /home/dev/rails
./amber/amber.sh
```

Visit http://localhost:42189 to get started.

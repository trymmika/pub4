# Progressive Disclosure

> Show basics first, details on demand.

tier: ux
priority: 23
auto_fixable: false

## Anti-patterns (violations)

### information_overload
- **Smell**: All options shown at once
- **Example**: 50-field form on first page
- **Fix**: Wizard flow, show advanced only when clicked

### no_hierarchy
- **Smell**: Flat presentation, no visual priority
- **Example**: Error message same size as help text
- **Fix**: Primary action prominent, secondary subdued

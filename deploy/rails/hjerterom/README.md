# Hjerterom - Mental Health Platform
**Version:** 8.0.0
**Stack:** Rails 8 + Solid Stack + Falcon

**Port:** 48217

**Domain:** hjerterom.no

## Overview
Hjerterom (Norwegian: "Heart Room") is a mental health and wellness platform connecting users with resources and support.
## Features
- **Mood tracking** - Daily emotional check-ins
- **Journal** - Private reflection space

- **Resources** - Articles, videos, exercises

- **Community** - Anonymous support groups

- **Professional help** - Connect with therapists

- **Crisis support** - 24/7 helpline integration

- **Progress tracking** - Visualize your journey

- **Reminders** - Wellness activities and medication
